# parrot-post-phishing-analysis
Project Overview

This repository documents a phishing email analysis exercise based on the Parrot Post scenario. The objective of this project is to identify, analyze, and document social engineering techniques used in a phishing email by examining both human-centric cues and technical email artifacts.

The exercise simulates a real-world phishing attack where an attacker impersonates a delivery service to trick users into clicking malicious links and submitting sensitive information.
